Misaki Anti Pédophile

Explication

❓C'est quoi la Brigade Anti P ?

La Brigade Anti P, de son vrai nom "Brigade Anti Pédo", est une team sur Misaki dont le but est d'identifier et de signaler les personnes suspectées d'être des pédophiles.

🎯Objectif

L'objectif est simple : faire disparaître les pédophiles de Misaki.

🪄Votre mission

Vous devrez vous rendre dans différents vocaux publics afin d'identifier les suspects, puis les signaler dans le salon ⛔・report. Le plus important : soyez actifs en vocal.

❔Comment ça marche ?

En tant que membre de la Brigade, vous avez accès aux commandes suivantes :

/report + ID du suspect + Raison + Preuves (screen ou enregistrement) — pour signaler une personne suspecte
/mineur + ID — pour enregistrer l'ID d'une personne mineure présente en vocal
/majeur + ID — pour enregistrer l'ID d'une personne majeure présente en vocal
/suspect + ID — quand vous êtes certains qu'une personne est suspecte ou a eu un comportement inquiétant

Si un mineur et un majeur enregistrés se retrouvent dans le même vocal, une logs automatique sera envoyée dans le salon dédié.

📹Enregistrements

Pour faire un enregistrement d'écran merci d'utiliser comme logiciel OBS STUDIO ou Medal

🏆Leaderboard

Pour récompenser les membres les plus investis, un classement a été mis en place. Vous avez accès à 2 classements : un pour le nombre de reports effectués, et un pour les heures passées en vocal sur Misaki.
